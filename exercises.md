## If exercises
### if and comparsion operations allowed only. no elif, else, and, or allowed
1. Write program that reads age from user and prints eligible in case he is above 18.
2. Write program that reads number from user and tells if number is even or odd.
3. Write program that reads age from user, in case age is between 18 and 30 with 18 and 30 included, prints eligible.
4. Write program that reads outside temperate,
   prints cold in case temperature is between 0 and 20 (0,20 included).
   prints moderate in case temperate is between 21 and 35 (21,35 included).
   prints hot in case temperate is 36 and above.
5. Write program that prompts user to select subject
```
   Please select your subject:
   Press M for Math
   Press P for Physics
   Press C for Computer Science
```
   Then prints the name of the selected subject `Math Selected`.
6. Extended the previous program. Ask for student's marks in the selected subject.
   Passing criteria for different subjects is different.
   70 and above for Math.
   60 and above for Physics.
   50 and above for Computer Science.
   Tell student if he/she passed or not.
7. Write a program that reads two different numbers from user and an operator
```
   Write 1st number: 5
   Write 2nd number: 6
   Write operator  : *
```
   that makes the expression as 5*6. Program solves the expression and prints the final result
8. Write a program that reads two numbers from the user and two operators
```
   Write 1st number: 2
   Write 2nd number: 9
   Write 1st operator: +
   Write 2nd operator: *
```
   Program selects the operator which has higher precedence, in the above case that is *, so the expression becomes 2*9.
